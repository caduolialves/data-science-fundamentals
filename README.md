# Data Science Fundamentals

Repositório dedicado ao estudo e prática dos **fundamentos de Ciência de Dados utilizando Python**.  
O objetivo deste projeto é documentar conceitos essenciais, exemplos práticos e pequenos experimentos com bibliotecas utilizadas no ecossistema de análise de dados.

Este repositório faz parte do meu **portfólio de aprendizado em Data Science**, contendo códigos, testes e explorações de bibliotecas como **NumPy, Pandas e Scikit-learn**.

---

# Objetivos do projeto

- Consolidar fundamentos de **Data Science**
- Praticar **manipulação e análise de dados em Python**
- Aprender boas práticas de organização de código
- Estudar algoritmos de **Machine Learning**

---

# Conteúdos abordados

O repositório inclui exercícios e exemplos relacionados a:

- Estruturas de dados em Python
- Manipulação de dados com **Pandas**
- Arrays e operações numéricas com **NumPy**
- Limpeza e preparação de dados (*data cleaning*)
- Exploração de datasets
- Importação e tratamento de arquivos (`csv`, `json`)
- Filtragem e transformação de DataFrames
- Implementação e estudo de algoritmos de **Machine Learning**

---

# Machine Learning

O repositório também contém estudos utilizando **Scikit-learn**, incluindo a implementação e testes do algoritmo:

- **K-Nearest Neighbors (KNN)**

O objetivo desses experimentos é entender:

- funcionamento do algoritmo
- preparação de dados para treinamento
- divisão entre **treino e teste**
- avaliação básica de modelos

# 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/caduolialves/data-science-fundamentals.git
```

## 🧪 Criando um ambiente virtual (recomendado)

Para evitar conflitos de dependências entre projetos, recomenda-se utilizar um **ambiente virtual (venv)**.

### 1. Criar o ambiente virtual

```bash
python -m venv .venv
```

### 2. Ativar o ambiente virtual

```bash
.venv\Scripts\activate
```

### 3. Instalar as dependências

```bash
pip install -r requirements.txt
```
